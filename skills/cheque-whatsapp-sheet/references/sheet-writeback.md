# Google Sheet 寫入規則

## 寫入前提
只有在以下條件同時成立時才可寫入：
- 所有必填欄位已補齊
- 原提交者已作清楚確認

## 固定結果回覆
- 寫入成功：`已成功儲存`
- 寫入失敗：`不成功儲存`

## 寫入失敗後續
如寫入失敗，下一則訊息必須補充具體原因，例如：
- Apps Script 回傳錯誤
- token 驗證失敗
- sheet 名稱錯誤
- 未取得寫入設定

不可只停留在「寫入唔到」。

## 敏感設定原則
- Google Sheet 連結、Apps Script Web App URL、token 等環境資料，不應硬編碼成可攜式 skill 的必要內容。
- 可攜式 skill 應保存流程與規則；實際 backend 設定應放在本地 workspace、部署文件或環境設定中。
