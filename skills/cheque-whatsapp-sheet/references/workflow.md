# 完整流程

## 輸入要求
最少必須包含：
- 門市編號
- 支票類別
- 相片

範例：
```text
PU003 鋪頭支票
```

## 主流程
1. 收到 WhatsApp 群組訊息。
2. 讀取支票／發票圖片。
3. 抽取欄位。
4. 用固定摘要格式回覆。
5. 等待原提交者確認。
6. 確認後寫入 Google Sheet。
7. 用固定結果字句回覆。

## 固定摘要回覆格式
```text
- 門市編號：PU003
- 支票類別：鋪頭支票
- 輸入日期：2026-03-27
- 支票付款日期：2026-03-24
- 支票號碼：067175
- 支票金額：HK$715.30
- 收款公司名：有盛行辦館有限公司
- Invoice No.：526021362
- 備註：-

請回覆：
- 1 = 確定並寫入 Google Sheet
- 2 = 有錯，需修改
```

## 資料不足時回覆格式
```text
以下資料未能確認：
- 支票付款日期
- Invoice No.

請補拍清楚右上角日期及發票編號。
```

## 寫入結果固定字句
- 成功：`已成功儲存`
- 失敗：`不成功儲存`
